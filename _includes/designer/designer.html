ORM Designer

Definitions on this website were modeled and generated ORM Designer, code generator and visualization tool for ORM frameworks.

Visualize your models

ORM Designer visualizes your models for you, easy design and editing:

  • Design the schema, no more painstakingly writing the definitions
  • Objects edited directly in the visual model
  • Diagram handles advanced ORM concepts like inheritance and ORM properies

Export your changes

Any changes to the model can be exported at will:

  • Repeatable editing and export
  • Export of standardized ORM Definitions
  • No typos or syntax errors

Import your projects

Supports quick and simple import of your models:

  • Powerfull import function
  • Support for XML, YAML and PHP code
  • Possibility to customize your data types, naming styles and predefined values
